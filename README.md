# Simple Calculator

A clean web calculator built with Next.js, React, Tailwind CSS, and vinext.

## Features

- Basic arithmetic: addition, subtraction, multiplication, division, and modulo
- Decimal and negative-number input
- Backspace, clear, and clear-history controls
- Calculation history panel
- Keyboard support for numbers, operators, Enter, Backspace, and Escape
- Responsive web interface

## Requirements

- Node.js `>=22.13.0`

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Useful Commands

- `npm run dev`: start local development
- `npm run build`: create a production build
- `npm run start`: run the production server
- `npm run lint`: run ESLint
